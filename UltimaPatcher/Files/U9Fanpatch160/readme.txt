u9fanpatch160.zip should contain 11 files:



readme.txt  -    this file

DialoguePatch.doc -    details about the dialogue patch

EconMonster.doc -    details about Novum Dragon's patches

text.flx -    modified U9 data file containing the new dialogue

typename.flx -    modified U9 data file containing the new item names

book -    modified (and renamed) U9 data file containing new books

book&economy -    modified (and renamed) U9 data file containing new books and Novum's Economy Patch

npc.flx -    modified U9 data file containing Novum's Monster Patch

u9name.exe -    used to select your Avatar name, turn on in-game text and turn off in-game speech

u9name.ini -    data file used with u9name.exe

The Pits of Carnage.doc -    unofficial prologue to the Dialogue Patch

Step-by-step Installation (same for first time install or upgrade)

Instruktionen fur die deutsche Version siehe unten

Note:  The following instructions assume you have installed Ultima IX into the default English language directory, c:\Program Files\Origin Systems\Ultima IX.  The location may vary on your system.


1. Unzip the files.

2. If you do not wish to install the Monster Patch, skip to step 3 now. If you do, copy "npc.flx" into c:\Program Files\Origin Systems\Ultima IX\runtime\.  The Monster Patch will not affect your existing savegames; you must start a new character for the changes to take effect.

3. If you do not wish to install the Economy Patch, skip to step 4 now.  If you do, rename "book&eco" to "books-en.flx" and copy it into c:\Program Files\Origin Systems\Ultima IX\static\ then SKIP TO STEP 5.

4. Rename "book" to "books-en.flx" and copy it into c:\Program Files\Origin Systems\Ultima IX\static\.

5. Copy "text.flx" into c:\Program Files\Origin Systems\Ultima IX\static\.

6. Copy "typename.flx" into c:\Program Files\Origin Systems\Ultima IX\static\.

7. Copy "u9name.exe" and "u9name.ini" into c:\Program Files\Origin Systems\Ultima IX\.  Run "u9name.exe" from the Ultima IX directory and select the name by which you will be known in Britannia. (If you decide you want to change your name again, repeat step 5 before running "u9name.exe").

8. Run Ultima IX and create a new character.  (If you're upgrading from an earlier version of the Dialogue Patch your old saves will work just fine, there's no need to start over from the beginning)




Patchinstallation für die deutsche Version

Als Standardinstallationsverzeichins wird hier c:\Programme\Origin Systems\Ultima IX genannt. Nutzt du ein anderes Verzeichnis für Ultima IX musst du dieses verwenden.

1.	Extrahiere die Dateien

2.	Wenn du den Monster Patch nicht installieren willst, fahre mir Schritt 3 fort. Wenn ja, kopiere "npc.flx" ins c:\Programme\Origin Systems\Ultima IX\static Verzeichnis. Du musst einen neuen Charakter starten, damit die neuen Lebensenergiewerte wirksam werden.

3.	Wenn du den Economy Patch nicht installieren willst, fahre mit Schritt 4 fort. Wenn ja, kopiere die Datei "book" ins c:\Programme\Origin Systems\Ultima IX\static Verzeichnis. Benenne die Datei in "books-de.flx" um. Fahre mit Schritt 5 fort.

4.	Kopiere die Datei "book&economy" ins c:\Programme\Origin Systems\Ultima IX\static Verzeichnis. Benenne die Datei in "books-de.flx" um.

5.	Kopiere die Datei "text.flx" ins c:\Programme\Origin Systems\Ultima IX\static Verzeichnis.

6.	Kopiere "u9name.exe" und "u9name.ini" ins c:\Programme\Origin Systems\Ultima IX Verzeichnis.

7.	Starte "u9name.exe" und gib den Namen deines Charakters ein.

8.	Kopiere "text.flx" im c:\Programme\Origin Systems\Ultima IX\static Verzeichnis und benenne sie in "text.de" um.

9.	Starte Ultima IX

Updates included in Version 1.6

1. Conversations in several areas have been touched up.

2. 1500 lines have had some minor changes (such as spaces, punctuation etc.).


Updates included in Version 1.5

1. Conversations in Minoc/Cove and Covetous have been touched up.

2. Several journal entries from the original dialogue that were inadvertently missed in earlier versions have been replaced to reflect the new dialogues.


Updates included in Version 1.4

1. Several new books added

2. Conversations in Moonglow and portions of Minoc have been touched up.

3. Miscellaneous minor changes have been made throughout the game (books, journal entries, conversations, etc.).


Updates included in Version 1.3

1. Various books added.

2. Various dialogues touched up (no changes to plot elements)

3. Bugs in Economy Patch which allowed the purchase of unequippable items have been fixed.

4. Corrections of various spelling, punctuation, and miscellaneous errors.


Updates included in Version 1.1

1. Readme.doc (Included in the 1.0 release) has been split into readme.txt and Dialogue Patch.doc to simplify the installation procedure.

2. Novum's Monster Patch has been added to the zip file to further simplify installation of the various fan patches available.

For further details regarding the various patches, please refer to the included Word documents.
